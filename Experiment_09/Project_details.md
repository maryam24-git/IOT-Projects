# Arduino ADC: Analog-to-Digital Conversion

This project demonstrates how to interface an analog sensor with an Arduino microcontroller to understand the fundamentals of Analog-to-Digital Conversion (ADC).

## Components Required
- **Arduino Uno R3**
- **Potentiometer** (used to simulate an analog sensor)
- **Jumper Wires**
- **Breadboard** (optional)

## Circuit Connections
Connect the components as follows:

| Potentiometer Pin | Arduino Pin |
| :--- | :--- |
| Terminal 1 | GND |
| Wiper (Middle Pin) | A0 |
| Terminal 2 | 5V |
