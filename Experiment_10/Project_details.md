## Temperature Sensor (LM35)

In this task, we interfaced an **LM35 Temperature Sensor** with an Arduino to measure the ambient temperature and display it in degrees Celsius (°C).

### Circuit Connections
The connections between the LM35 sensor and the Arduino are as follows:

| LM35 Pin | Arduino Pin |
| :--- | :--- |
| **Vs (Pin 1)** | 5V |
| **Vout (Pin 2)** | A1 |
| **GND (Pin 3)** | GND |
